
FIRE FIGHTING ROBOT - WIRING GUIDE

========================================
1) ARDUINO UNO CONNECTIONS
========================================

FLAME SENSORS:
- Left sensor  -> A0
- Center sensor-> A1
- Right sensor -> A2

ULTRASONIC SENSOR (HC-SR04):
- VCC -> 5V
- GND -> GND
- TRIG -> D12
- ECHO -> D13

RELAY (PUMP):
- IN -> D7
- VCC -> 5V
- GND -> GND

========================================
2) MOTOR DRIVER (L298N)
========================================

RIGHT MOTOR:
- IN1 -> D8
- IN2 -> D9

LEFT MOTOR:
- IN3 -> D10
- IN4 -> D11

PWM SPEED CONTROL:
- ENA -> D5
- ENB -> D6

POWER:
- 12V -> Battery +
- GND -> Battery -

Arduino GND MUST connect with L298N GND

========================================
3) POWER NOTES
========================================
- Use separate battery for motors (recommended 7V-12V)
- Arduino powered separately or via regulator
- All grounds must be common

========================================
4) SIMPLE FLOW
========================================
1. Read flame sensors
2. Determine direction
3. Move robot
4. Check ultrasonic distance
5. If distance <= 10cm -> stop + pump ON
6. Else -> keep moving

========================================
5) IMPORTANT NOTES
========================================
- Keep ultrasonic facing forward
- Keep flame sensors spaced left/center/right
- Do NOT power motors from Arduino 5V
